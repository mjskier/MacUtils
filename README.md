# MacUtils
Various scripts developed to administer Macs
* cid (Change user ID)

   In a mixed Mac and Linux environment, it makes it a lot easier if User IDs match accross systems.
   This utility changes the ID of a given user to a given ID.
   
   ```cid -u <user> -n <new ID>```
